# hjonk-windows-notifier
basically all this does is give you a notification if someone from the users.txt file has posted

## How to use
1. Clone the repo
2. Do ```pip install -r requirements.txt```
3. Create users.txt file with the usernames of the people you want notifications from (without @)
4. Run main.py

## Example users.txt
```
dumbina
applefritter
sewfews
```
thats it
